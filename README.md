# Flamma Digital – Agency Website

This repository contains the source code for the Flamma Digital agency website. It is a custom-built landing page designed to showcase our web development services and Shopify eCommerce solutions.

## About the Project

Flamma Digital specializes in developing custom Shopify themes and tailored code solutions. This website serves as a landing page that explains our services, highlights the benefits of using Shopify storefronts, and presents the features we offer.

Through the site, potential clients can learn more about our work, explore our services, and contact us directly to request project quotes.

## Built With

* HTML5
* CSS3
* Vanilla JavaScript
* Hosted on Netlify
